## PS-TESTER 개요

- 추진계통 시험장치(Propulsion System TESTER)는 무인항공기(드론)의 추력 시스템 개발에서 모델링, 시험 및 설계를 위하여 정량적으로 측정하는 프로펠러-모터 추력 시험 장치임.

<u>그림. PS-TESTER 시스템 Overview</u>

- 데이터 수집 장치로 EDAS(Engineering Data Acquisition System, 공학자료수집장치) 사용
	- USB 인터페이스 통한 제어 및 모니터링, 데이터 로깅(logging) 제공
	- 정확하고 정밀한 데이터의 수집 및 관리 수행 – 추력과 토크 정밀도 0.1% FS

- 프로펠러의 입력 동력(Dynamic Power)에 대한 추력과 토크의 정밀 측정으로 성능 시험
- 프로펠러, 모터, 전원(배터리) 작동에 따른 입력 전력(Electric Power), 온도, 공기밀도 등의 시험 환경 측정으로 추진계통 특성의 정량화와 신뢰성 확보
- 프로펠러의 추력계수과 동력계수로 환산하여 드론 설계 데이터 제공

## PS-TESTER 특징

- 1000ms/100ms/10ms 제어 및 데이터 수집(Logging) 간격으로 Manual/Auto 수행
- 고정밀 2축 로드셀로 추력과 토크 측정 – 오차 0.1%(추력/토크)
- 고정밀 데이터 수집 장치 EDAS와 열전대 온도 측정장치(TC-TEMP)는 자사 제품 사용
- Function Signal과 Profile List를 이용한 제어 및 측정 데이터 수집
	- Manual/Auto Profile과 Slider에 의한 PWM 생성
	- 신호 함수(사인파, 삼각파, 계단파)에 의한 PWM 생성

## PS-TESTER Manager 주요 기능

- Operation Mode 표시 - PWM(%), Speed(rpm), 통신상태, Logging 상태
- Operation Mode - Signal Operation / Manual Profile / Automatic Profile / Testing Operation / PWM Control Slider
- Data Logging - 로깅 주파수(Hz) 설정 (1/10/100 Hz)
- Realtime Graph  - 실시간 데이터 그래프
- 정지 추력계수(CT0)/정지 전력계수(CP0)
- 실시간 측정 데이터 - Power/회전수/추력/토크/전류/전압/온도/대기정보 표시

## PS-TESTER 측정 데이터 및 분석 

- 프로펠러의 추력과 토크
- 모터의 온도와 회전속도(RPM)
- 대기압과 대기온도 – 공기 밀도 계산
- 모터 입력 전압과 전류 – 입력 전력
- 배터리 온도, 전압 – 모터까지의 전압강하
- 프로펠러 동력

- 추진 시스템의 요소별 효율 추산
- 추진 효율의 최적화
